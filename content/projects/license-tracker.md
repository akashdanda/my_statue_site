title: LicenseTracker
date: 2025-08-01
description: Fine-tuned YOLOv11 for license plate detection with real-time recognition system
technologies: YOLOv11, AWS Textract, Streamlit, AWS EC2, Roboflow
---

Fine-tuned YOLOv11 on 10k+ license plate images, achieving mAP@50 of 0.96 for high-accuracy detection. Built real-time recognition system using AWS Textract and Streamlit for OCR-based output. Deployed system on AWS EC2 with image storage and retrieval through AWS S3 buckets for scalability.
